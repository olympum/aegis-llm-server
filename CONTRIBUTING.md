# Contributing to Aegis LLM Server

Thank you for your interest in contributing to Aegis LLM Server.

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). By participating, you agree to uphold this code.

## How to Contribute

### Reporting Issues

- Check existing issues before opening a new one.
- Include reproduction steps, expected behavior, and actual behavior.
- Include relevant version and environment details.

### Pull Requests

1. Fork the repository.
2. Create a feature branch (`git checkout -b feature/amazing-change`).
3. Make your changes.
4. Add or update tests for behavior changes.
5. Run tests and lint locally.
6. Commit your changes (`git commit -m 'feat: add amazing change'`).
7. Push your branch and open a Pull Request.

## Commit Policy

- `Co-authored-by:` trailers are not allowed in this repository.
